== Materials and boundary conditions

=== Materials

The beam is made of steel, so we need to use the following data:

* E = 200 GPa -> 200e3 N/mm^2^
* nu = 0.3
* rho = 7870 kg/m^3^ -> 7870e-9 kg/mm^3^

=== Boundary conditions
